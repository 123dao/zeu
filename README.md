## Usage

#### Digital Clock

```javascript
var canvas = document.getElementById('canvas-id');
var digitalClock1 = new zeu.DigitalClcok(
  canvas,
  {
    dashColor: '#DCDCDC',
    barWidth: 8,
    barHeight : 30,
    space: 10
  }
);
digitalClock1.draw();
```
